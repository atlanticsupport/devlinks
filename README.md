# DevLinks 🔗

Uma aplicação para gerir e partilhar links úteis para developers.

## Stack
- **Frontend**: Next.js + React
- **Backend**: Supabase (PostgreSQL)
- **Deploy**: Vercel
- **Repo**: GitHub

## Features
- 📚 Organizar links por categorias
- ⭐ Marcar favoritos
- 🏷️ Sistema de tags
- 👥 Partilha em equipa
- 🔍 Pesquisa rápida

## Setup
```bash
npm install
npm run dev
```
